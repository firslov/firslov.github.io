Hugo+github自建blog流程

  1. brew install >>>hugo 安装
  2. hugo new site mysite >>>创建一个网站
  3. cd themes/ git clone … >>>下载主题
  4. 配置 config.toml
  5. hugo post/xxx.md >>>新建文章，目录是 mysite/content
  6. hugo server -t 主题名 -D >>>本地预览 http://localhost:1313/
  7. 根目录下 hugo >>>生成静态网站，目录 /public
  8. push public/ 全部文件到 github.io
